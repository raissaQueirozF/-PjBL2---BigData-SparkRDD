# Big Data com Apache Spark

Projeto desenvolvido em Python utilizando Apache Spark para processamento e análise de mais de 8 milhões de transações comerciais internacionais. O trabalho aplica operações com RDD e PairRDD para consultas estatísticas, filtragens, agregações e análise de exportações, preços e categorias de produtos ao longo de 30 anos de dados.

## Funcionalidades
- Processamento de grandes volumes de dados em CSV
- Tratamento de cabeçalho e dados faltantes
- Consultas utilizando RDD e PairRDD
- Cálculo de médias, máximos e mínimos
- Análise de exportações por país
- Ordenação e agrupamento de dados por ano, flow e categoria
- Identificação de transações com maior preço por kg

## Tecnologias
- Python
- Apache Spark
- PySpark

## Dataset
Base contendo mais de 8 milhões de registros de transações comerciais internacionais, incluindo:
- País
- Ano
- Categoria
- Tipo de transação (Import/Export)
- Preço
- Peso
- Quantidade

## Objetivo
Demonstrar o uso de computação distribuída com Spark para análise eficiente de grandes volumes de dados.
